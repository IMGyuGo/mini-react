# Tests

현재는 자동 테스트 전 단계의 스텁 파일이 포함되어 있습니다.

## 포함된 스텁

- `framework.test.html` — 브라우저에서 보는 테스트 시나리오 목록
- `component.test.js` — 컴포넌트 관련 검증 항목 스텁
- `hooks.test.js` — Hook 관련 검증 항목 스텁
- `integration.test.js` — 통합 시나리오 스텁

## 다음 단계

- 각 스텁 시나리오를 실제 assertion 테스트로 교체
- framework 구현 완료 후 브라우저/모듈 테스트 실행 환경 연결
